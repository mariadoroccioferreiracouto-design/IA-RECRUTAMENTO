# IA Recrutamento Teste

Sistema de entrevista automatizada utilizando Inteligência Artificial aplicada ao setor de Recursos Humanos.

## 📌 Objetivo

O projeto tem como objetivo simular um processo de recrutamento e seleção automatizado, avaliando candidatos por meio de perguntas estruturadas e gerando um relatório com pontuação baseada em critérios pré-definidos.

## 🚀 Funcionalidades

- Entrevista automatizada em formato de chat
- Avaliação baseada em critérios técnicos e comportamentais
- Geração de relatório final com pontuação
- Barra visual de desempenho
- Exportação dos dados do candidato em formato JSON

## 💼 Aplicação Acadêmica

Este sistema foi desenvolvido como protótipo para estudo do impacto da Inteligência Artificial no setor de Recursos Humanos, com foco em recrutamento e seleção.

## 🛠 Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
